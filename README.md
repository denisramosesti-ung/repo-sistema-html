# Hospital San Martín (demo)
Sitio estático para práctica de cátedra (HTML5, JS, CSS).

## Estructura
- `index.html` — portada
- `servicios.html` — servicios + links a PDF
- `turnos.html` — calendario y reserva simulada (localStorage)
- `contacto.html` — formulario mailto
- `assets/` — CSS, JS, imágenes
- `docs/` — PDFs descargables

## Cómo probar localmente
Abrí `index.html` en el navegador. No necesita backend.

## Despliegue en GitHub Pages
1. Subí este contenido a un repo nuevo (por ejemplo `hospital-web`).
2. En GitHub: **Settings → Pages → Build and deployment**.
3. En **Source**, elegí **Deploy from a branch**.
4. En **Branch**, seleccioná `main` y la carpeta `/root`.
5. Guardá y esperá a que se publique. La URL será `https://<tu-usuario>.github.io/hospital-web/`.
